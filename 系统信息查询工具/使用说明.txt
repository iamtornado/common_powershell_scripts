系统信息查询工具 - 使用说明
=====================================

📋 工具简介
-----------
本工具专为企业员工设计，用于快速获取Windows计算机的基本系统信息，
方便IT运维工程师进行故障诊断和技术支持。

📁 文件说明
-----------
1. Get-BasicInfo.ps1      - 基础信息查询脚本（推荐使用）
2. Get-SystemInfo.ps1     - 详细信息查询脚本
3. 运行脚本.bat           - 一键运行工具（最简单的使用方式）
4. 使用说明.txt           - 本说明文件

🚀 使用方法
-----------

方法一：使用批处理文件（推荐）
1. 双击 "运行脚本.bat" 文件
2. 根据提示选择需要的功能：
   - 选择 1：快速获取核心信息（推荐）
   - 选择 2：获取详细系统信息
   - 选择 3：获取详细信息并保存到文件
3. 按任意键查看结果

方法二：直接运行PowerShell脚本
1. 右键点击PowerShell脚本文件
2. 选择"使用PowerShell运行"
3. 如果出现执行策略错误，请以管理员身份运行PowerShell，
   然后执行：Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

📊 输出信息说明
---------------
基础信息查询包含：
✓ 计算机名（FQDN）- 计算机的完整域名
✓ 当前登录用户 - 当前使用电脑的用户账户
✓ IP地址 - 计算机在网络中的地址（仅显示已连接的网卡）
✓ MAC地址 - 网络适配器的物理地址

详细信息查询额外包含：
✓ 操作系统版本和架构
✓ 计算机制造商和型号
✓ 总内存容量
✓ 所有活动网络适配器的详细信息

🎯 使用场景
-----------
当您遇到以下情况时，请运行此工具并将结果提供给IT工程师：

1. 电脑出现网络连接问题
2. 需要远程技术支持
3. 软件安装或配置问题
4. 系统故障诊断
5. 资产管理和盘点

💡 使用提示
-----------
1. 建议使用"基础信息查询"，信息简洁明了
2. 如需保存信息，选择"详细信息查询并保存到文件"
3. 生成的SystemInfo.txt文件可以通过邮件发送给IT工程师
4. 脚本运行完毕后会暂停，按任意键可退出
5. 如果脚本无法运行，请联系IT管理员检查PowerShell执行策略

⚠️ 注意事项
-----------
1. 本工具仅获取基本系统信息，不会修改任何系统设置
2. 所有信息均为只读获取，对系统安全无影响
3. 如果网络适配器显示为空，可能是网络连接异常
4. 某些企业环境可能限制PowerShell脚本执行，请联系IT管理员

🔧 故障排除
-----------
问题：双击批处理文件无反应
解决：右键选择"以管理员身份运行"

问题：PowerShell脚本执行被阻止
解决：以管理员身份运行PowerShell，执行以下命令：
     Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

问题：显示乱码
解决：确保系统支持UTF-8编码，或联系IT管理员
